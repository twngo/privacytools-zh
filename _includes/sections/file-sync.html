值得一提

  • git-annex - 用 git 來管理檔案，但不用 git 來檢查檔案內容。這聽起來似乎有點矛盾，但當在處理大型檔案上這方法就很好用，尤其是考慮到記憶體大小、時間和硬碟空間。
